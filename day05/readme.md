# Number Guessing Game

This is a number guessing game written in Python.

The computer chooses a random number between 1 and 20, and the player has to guess it.

---

## Installation

No external dependencies are required.

All you need is:
- Python 3 installed on your computer

## How to Play

- Enter a number between 1 and 20 to make a guess.
- If your guess is too low, the game will say "too small".
- If your guess is too high, the game will say "too big".
- If you guess correctly, you will win.


## Special Commands (By Level)

- `x` – Exit the game  
  (Available from Level 2 and onward)

- `s` – Show the hidden number once (cheat)  
  (Available from Level 3 and onward)

- `d` – Toggle debug mode  
  Shows the hidden number before each guess when enabled  
  (Available from Level 4 and onward)

- `m` – Toggle move mode  
  The hidden number changes slightly after each guess  
  (Available from Level 5 and onward)

- `n` – Start a new game with a new hidden number  
  (Available in Level 6)
